# homebrew-nexum

Homebrew tap for [Nexum](https://github.com/Snenjih/nexum) — a macOS-native remote workspace with SSH, SFTP, Terminal, Editor, and AI.

## Install

```sh
brew tap snenjih/nexum
brew install --cask nexum
```

> **Note:** Nexum is not yet notarized. macOS may show a security warning on first launch.
> To open the app: right-click `Nexum.app` in `/Applications` → **Open** → **Open** again in the dialog.

## Update

```sh
brew upgrade --cask nexum
```

## Uninstall

```sh
brew uninstall --cask nexum
brew untap snenjih/nexum
```
